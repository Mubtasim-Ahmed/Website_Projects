# Beginner Small Responsive Projects
A collection of beginner-friendly responsive frontend projects built using HTML, CSS, and JavaScript.
-


### 01. Digital Clock
A simple responsive digital clock application that displays the current local time and date in real time.

🌐 Live Preview - https://mub-digitalclock.netlify.app/


### 02. Calculator App
A basic calculator application capable of performing simple arithmetic operations with a clean user interface.
### Features
- Addition
- Subtraction
- Multiplication
- Division

🌐 Live Preview - https://mub-calculator.netlify.app/

### 04. BMI Calculator
A responsive BMI (Body Mass Index) Calculator built using HTML, CSS, and JavaScript.  
This application allows users to calculate their BMI using either centimeters or feet & inches measurement units.

### Features
- Calculate BMI instantly
- Supports:
  - Centimeters
  - Feet & Inches
- Responsive modern UI
- BMI category indicator
